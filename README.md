NutshellCrumpy
==============

Python wrapper for Nutshell CRM API

<h3>Some basic examples:</h3>
<body>

from NutshellCrumpy import Nutshell
n=Nutshell(my_email, my_api_key)
<br>
n.findTags()
<br>
n.response
<br>
<br>
'{"result":{"Leads":[],"Contacts":[],"Accounts":["Some new tag"]},"id":"apeye","error":null}'
